> The Simple Next CRUD APP is a project that will help you learn a litle about React/Next, Typescript and Tailwind CSS.

## Prerequisites

- React: 17.0.2
- Next: 12.0.7
- Typescript: last version
- Firebase: 8.8.0  


## Install

```sh
npm install
```

## Usage

```sh
npm run dev
```
